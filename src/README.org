# Optimal Transport on Graphs

Quick start for working on the project:

First of all, make sure that Julia is installed, along with the BlockBandedMatrices, CUDA, and ProgressMeter packages.
I would also recommend the Revise package so that you can edit and test functions without leaving your julia session.

Example work flow

1. start julia
2. execute the following
#+BEGIN_JULIA:
using Revise
includet("EarthMover.jl")
Q = [0. 1.; 1. 0.]; a = [2.0; 0]; b = [0.; 2]; N = 100;
vector, dist = BBD(Q, a, b, N)
#+END_JULIA

Testing individual files

#+BEGIN_JULIA:
using Revise
includet("src/KProjection.jl")
includet("tests/KProjectionTest.jl")
test_proj_ktop()
#+END_JULIA:
